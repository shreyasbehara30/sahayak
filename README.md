# Sahayak: AI Assistant for Offline Classrooms

![Sahayak Banner](/web/public/sahayak-logo.jpeg)

**Sahayak** is an offline-first AI classroom assistant designed to empower teachers in rural schools. Built for Android tablets, it provides multilingual lesson plans, voice-based doubt resolution, and visual learning aids without requiring an active internet connection.

## 🚀 Key Features

- **Offline-First Architecture**: Core functionality works entirely without internet access.
- **Multilingual Support**: Generates lesson plans and content in local languages.
- **Voice Assistant**: Voice-enabled interface for hands-free queries during class.
- **Visual Learning**: AI-generated visual aids to explain complex concepts.
- **Teacher Dashboard**: Tools for classroom management and curriculum tracking.
- **Low-Resource Optimized**: Optimized to run smoothly on budget Android tablets (4GB RAM+).

## 🛠️ Tech Stack

- **Frontend**: [Next.js](https://nextjs.org/) (React Framework)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: Framer Motion, Vanta.js
- **Icons**: Lucide React

## 📦 Installation (Development)

To run the web dashboard/landing page locally:

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/shreyasbehara30/sahayak.git](https://github.com/shreyasbehara30/sahayak.git)
    cd sahayak/web
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in Browser**:
    Visit https://sahayak-kappa.vercel.app/ to see the application.
